## Networking Mini-Project 2: DNS & Firewall Simulator

### Goal
Simulate DNS resolution and firewall rules for a domain and multiple ports.

### Script Overview
- Resolve a domain name to an IP address
- Check if each port is allowed or blocked by the firewall
- Uses arrays, loops, and conditionals in Bash

### Concepts Practiced
- DNS resolution (domain → IP)
- Firewall rules
- IP addresses and ports
- Bash arrays and loops
- Conditional statements (if/elif/else)
